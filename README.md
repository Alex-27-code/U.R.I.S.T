# Ю.Р.И.С.Т.

Веб-платформа для записи к юристу + Telegram-бот. Клиенты выбирают специалиста и бронируют время, юристы управляют расписанием, администратор контролирует всё через панель.

**Стек:** Python, Flask, SQLite, aiogram

## Запуск

```bash
pip install -r requirements.txt
python main.py
```

## Telegram-бот

Бот использует ту же базу данных, что и сайт (`data/urist.db`).
Чтобы указать другой путь, задайте переменную окружения `URIST_DB_PATH`.

## Обратная связь

Для отправки писем задайте переменные окружения:

```
MAIL_USER=ваш_gmail@gmail.com
MAIL_PASS=пароль_приложения_google
```
